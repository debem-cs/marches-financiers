# Model and definition

I Consider a financial market consisting of $d = 1$ risky asset, defined onsome probability space $( \Omega , \mathcal { F } , \mathbb { P } )$ endowed with filtration$\mathbb { F } = ( \mathcal { F } _ { t } ) _ { 0 \leq t \leq T }$ . For simplicity, we may assume ${ \mathcal { F } } _ { 0 } = \{ \emptyset , \Omega \}$ and$\mathcal { F } _ { T } = \mathcal { F }$ .

I We focus on the pricing of American options. Recall that the holdercan exercise the American optional at any time $t \in \{ 0 , \ldots , T \}$ . If theoption is exercised at t, the holder will receive a payoff equal to $H _ { t }$ ,where $H _ { t }$ is a positive v.a. that is $\mathcal { F } _ { t }$ −measurable. The option canonly be exercised once.

I Example : the American put option of strike K on the risky asset isdefined by $H _ { t } = ( K - S _ { t } ) ^ { + }$ .

I We assume the market is arbitrage-free and complete. We denote by$\mathbb { Q }$ the equivalent martingale measure.

# Snell envelope

Let $( \Omega , \mathcal { F } , \mathbb { P } )$ be a probability space with filtration $\scriptstyle ( { \mathcal { F } } _ { n } ) _ { n = 0 , \ldots , N } .$

Let $\scriptstyle ( Z _ { n } ) _ { n = 0 , \ldots , N }$ an $\mathbb { F }$ -adapted positive process.

# Definition

The Snell envelope of $\left( Z _ { n } \right) _ { n }$ is the process $\left( E _ { n } \right)$ defined by

$$
\left\{ \begin{array}{l} E _ {N} = Z _ {N}, \\ E _ {n} = \max  \left(Z _ {n}, \mathbb {E} \left[ E _ {n + 1} | \mathcal {F} _ {n} \right]\right), \quad n = 0, \ldots , N - 1. \end{array} \right.
$$

This is the smallest supermartingale dominating from above $\left( Z _ { n } \right) _ { n }$

# Proposition

Set $\nu _ { 0 } = \mathfrak { i n f } \{ n \geq 0 : E _ { n } = Z _ { n } \}$ . Then ν0 is a stopping time and thestopped process $\left( E _ { n \wedge \nu _ { 0 } } \right) _ { n = 0 , \dots , N }$ is martingale.

# American vs European

# Proposition

Denote by $V _ { t } ^ { a }$ the value of American option at t of payoff (H0, . . . , HT ).Denote by $V _ { t } ^ { e }$ the value of European option at t of payoff HT . We have

(a) $V _ { t } ^ { a } \geq V _ { t } ^ { e }$ for all $t \in \{ 0 , \ldots , T \}$ ;

(b) si $H _ { t } \leq V _ { t } ^ { e }$ for all t, then $V _ { t } ^ { a } = V _ { t } ^ { e }$ for all t .

# Corollary

For the case of call option, i.e. $H _ { u } = ( S _ { u } - K ) _ { + }$ and r ≥ 0, we have$V _ { t } ^ { a } = V _ { t } ^ { e }$ for all t .

I The last result is no longer true for the put option, if we adddividends, etc.

# Replication of American options

# Proposition

Let $U _ { t }$ be the value of American option at t of payoff (H0, . . . , HT ).Assume that the market is complete. Then :

(i) there exists a strategy $( \xi _ { t } ^ { 0 } , \xi _ { t } )$ whose value process is $V _ { t }$ s.t.$V _ { t } = U _ { t } + A _ { t }$ , where A is a predictable and non-decreasing processwith $A _ { 0 } = 0$ ;

(ii) On the behalf of seller, a perfect replication existe ;

(iii) On the behalf of buyer, the optimal stopping time τ is determined by$H _ { \tau } = U _ { \tau }$ and τ ≤ inf{k ≥ 0 : Ak+1 > 0} ;

(iv) A no optimal exercise time of buyer leads to a gain of seller whoseadopts the perfect replication strategy.
