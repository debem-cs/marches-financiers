# Stochastic process

Let $( \Omega , \mathcal { F } , \mathbb { P } )$ be a probability space.

# Definition (stochastic process)

A discrete-time stochastic process is a family of r.v.s $X = ( X _ { t } ) _ { t \in I }$ definedon $( \Omega , { \mathcal { F } } )$ and taking values in Rd , where $I = \{ 0 , 1 , \ldots , T \}$ is the indexset with T ∈ N or $T = \infty$ .

▶ Throughout this course we consider the discrete-time setting$I = \{ 0 , 1 , \ldots , T \} .$

▶ For the continuous-time case $I = [ 0 , T ]$ or $I = \mathbb { R }$ , see e.g. “CalculStochastique 3A”.

# Head or tail

▶ Probability $\begin{array} { r } { \mathtt { s p a c e : } \Omega : = E ^ { T } = \{ \omega = ( \omega _ { 1 } , \dots , \omega _ { T } ) : } \\ { \{ A : A \subset \Omega \} \mathtt { a n d } \mathbb { P } ( \omega ) : = 1 / | \Omega | = 1 / 2 ^ { T } } \end{array}$ ωi ∈ E },${ \mathcal { F } } : = 2 ^ { \Omega } = \{ A : A \subset \Omega \}$ .

▶ Let $X _ { 1 } , \ldots , X _ { T }$ be r.v.s defined by $X _ { i } ( \omega ) = 1$ if ωi = head and$X _ { i } ( \omega ) = 0$ if $\omega _ { j } = t a i I$ . Then $( X _ { i } ) _ { 1 \leq i \leq T }$ is a stochastic process takingvalues in $\{ 0 , 1 \}$ .

▶ Let $S _ { t } : = \sum _ { i = 1 } ^ { t } X _ { i }$ . Then $( S _ { n } ) _ { 1 \leq t \leq T }$ is a stochastic process takingvalues in {0, . . . , T }.

# Filtration

# Definition (Filtration)

A filtration is an increasing sequence of sub-algebras $( { \mathcal { F } } _ { t } ) _ { t \in I }$ of $\mathcal { F }$ , i.e.

$$
\mathcal {F} _ {t} \subseteq \mathcal {F} _ {t + 1} \subseteq \mathcal {F}, \quad \forall 0 \leq t \leq T - 1.
$$

▶ Recall that $E = \{ h e a d , t a i I \}$ , Ω = E T and ${ \mathcal { F } } = 2 ^ { \Omega }$

Set $\mathcal { F } _ { 0 } : = \{ \varnothing , \Omega \}$ and $\mathcal { F } _ { t } : = \sigma ( X _ { 1 } , \ldots , X _ { t } )$ for 1 ≤ t ≤ T .

▶ $( \mathcal { F } _ { t } ) _ { 0 \leq t \leq T }$ is a filtration for $( \Omega , \mathcal { F } , \mathbb { P } )$

$\mathcal { F } _ { t } = \sigma ( S _ { 1 } , \ldots , S _ { t } )$ for 1 ≤ t ≤ T .

$\mathcal { F } _ { t }$ stands for the accumulated information up to time t.

# Adapted/Predictable process

# Definition (adapted/predictable process)

X is called adapted (resp. predictable) w.r.t. to filtration $( { \mathcal { F } } _ { t } ) _ { t \in I }$ if $X _ { t }$(resp. $X _ { t + 1 } )$ ) is $\mathcal { F } _ { t }$ −measurable for all $t \in I$ .

▶ Adapted processes are used to model the events to come out, likeprices.

▶ Predictable processes are used to model a decision made for the nearfuture $t + 1$ , provided the information up to time t, like investingstrategies.

# Proposition

If X is adapted, then $X _ { s }$ is $\mathcal { F } _ { t }$ −measurable for all s ≤ t.

# Generated filtration

# Definition

We call FX the filtration generated by X which is defined as

$$
\mathcal {F} _ {t} ^ {X} := \sigma (X _ {0}, \dots , X _ {t}).
$$

# Proposition

X is adapted w.r.t. its generated filtration.

▶ F X is the smallest filtration that makes X adapted.

# Stopping time

# Definition (stopping time)

A random variable $\tau$ taking values in $\mathbb { N } \cup \{ \infty \}$ is called a stopping time if$\{ \tau \leq t \} \in \mathcal { F } _ { t }$ for all $t \in \mathbb { N }$ .

# Proposition

▶ A random variable $\tau$ taking values $i n \ N \cup \{ \infty \}$ is a stopping time ifand only $i f \left\{ \tau = t \right\} \in \mathcal { F } _ { t }$ for all $t \in \mathbb { N }$ .

▶ Let $A \in B ( \mathbb { R } ^ { d } )$ and X be adapted. Then the first hitting time$\tau : = \operatorname* { i n f } \{ t \in I : X _ { t } \in A \}$ is a stopping time, where inf $\varnothing : = \infty$ .

# Discrete-time martingale

Let $( X _ { t } ) _ { t \in I }$ be adapted and integrable, i.e. $X _ { t }$ is $\mathcal { F } _ { t }$ -measurable and$\mathbb { E } [ | X _ { t } | ] < \infty$ for all $t \in I$ .

# Definition (martingale)

X is called

(i) sub-martingale if $\mathbb { E } [ X _ { t } | { \mathcal F } _ { s } ] \ge X _ { s }$ a.s. for all $t , s \in I$ such that $t \geq s$ ;

(ii) super-martingale i $\mathsf { f } \mathbb { E } [ X _ { t } | \mathcal { F } _ { s } ] \leq X _ { s }$ a.s. for all $t , s \in I$ such that $t \geq s$ ;

(iii) martingale if $\mathbb { E } [ X _ { t } | \mathcal { F } _ { s } ] = X _ { s }$ a.s. for all $t , s \in I$ such that $t \geq s$

It is enough to show the claims with $t = s + 1$ for all $s \in I$ (such that$s + 1 \in I )$ . In the case of a martingale and if $I = \{ 0 , \ldots , T \}$ it is enoughto prove that $\mathbb { E } [ X _ { T } | \mathcal { F } _ { s } ] = X _ { s }$ for all $s \in I$ .

# Proposition

If X is martingale, $\mathbb { E } [ X _ { t } ] = \mathbb { E } [ X _ { 0 } ]$ for all t ∈ I . An adapted and integrableprocess X is martingale iff E[Xτ ] = E[X0] for all bounded stopping times τ .

# Doob’s martingale inequality

# Theorem

$$
\mathbb {P} \left(\max  _ {1 \leq i \leq n} X _ {i} > x\right) \leq \frac {\mathbb {E} \left[ X _ {n} ^ {+} \right]}{x}.
$$

\mathbb{P}\big(\max_{1\leq i\leq n}X_{i} > x\big)\leq \frac{\mathbb{E}[|X_{n}|^{p}]}{|x|^{p}}
