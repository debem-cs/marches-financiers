# Equivalent probability measures

Let $\mathbb { P } , \mathbb { Q }$ be two probability measures on $( \Omega , { \mathcal { F } } )$ .

# Definition (Absolutely continuous probability measures)

$\mathbb { Q }$ is absolument continue w.r.t. P, denoted by $\mathbb { Q } \ll \mathbb { P }$ if and only if for all$A \in { \mathcal { F } } \operatorname { \mathbb { P } } ( A ) = 0 \Longrightarrow \operatorname { \mathbb { Q } } ( A ) = 0 .$

# Definition

$\mathbb { P }$ and $\mathbb { Q }$ are equivalent if $\mathbb { Q } \ll \mathbb { P }$ and $\mathbb { P } \ll \mathbb { Q }$ . This is denoted by $\mathbb { P } \sim \mathbb { Q }$ .The two measures have the same impossible (resp. sure) events.

Let $\mathbb { Q }$ be a probability measure with density $Z$ with respect to $\mathbb { P }$ then$\mathbb { Q } \ll \mathbb { P }$ . Indeed let $A \in { \mathcal { F } }$ such that $\mathbb { P } [ A ] = 0$ then

$$
\mathbb {Q} [ A ] = \int 1 _ {A} Z d \mathbb {P} = 0.
$$

Assume further $\mathbb { P } [ Z > 0 ] = 1$ , then $\mathbb { P }$ and $\mathbb { Q }$ are equivalent.

# Radon-Nykodym Theorem

# Theorem

Let Q and P be two probability measures on $( \Omega , { \mathcal { F } } )$ . Then the followingassertions are equivalent

1. $\mathbb { Q } \ll \mathbb { P }$ .

2. There exists $f \in L ^ { 1 } ( \Omega , \mathcal { F } , \mathbb { P } )$ f ≥ 0 (unique a.s.) such that Q has adensity function $f$ with respect to $\mathbb { P }$ , i.e. $\mathbb { Q } = \boldsymbol { \mathscr { f } } \cdot \mathbb { P }$ or

$$
\mathbb {Q} [ A ] = \int 1 _ {A} f d \mathbb {P} = \mathbb {E} [ 1 _ {A} f ] o r \frac {d \mathbb {Q}}{d \mathbb {P}} = f.
$$
