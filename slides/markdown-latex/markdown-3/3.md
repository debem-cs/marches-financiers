# Assets I

▶ Consider a market of one period (or equivalently two dates) : 0 now,and $T = 1$ future.

$$

# Arbitrage

# Definition

We say that a portfolio $( \xi ^ { 0 } , \xi )$ yields an arbitrage opportunity if

(i) V ξ 0 ,ξ = 0,

(ii) V ξ0,ξ ≥ 0 P-a.s.,

(iii) P[V ξ0,ξ > 0] > 0.

▶ The existence of an arbitrage is perceived as a market inefficiency,which will be exploited and quickly disappears.

▶ When there is no arbitrage, the market is called arbitrage free orefficient.

# Payoff

# Definition

A payoff is identified by a r.v. on $( \Omega , \mathcal { F } _ { 1 } , \mathbb { P } )$ .

Interpretation : payment depending on the scenario ω at time t = 1.

# Definition

The space $\{ \xi ^ { 0 } S _ { 1 } ^ { 0 } + \xi \cdot S _ { 1 } : \xi ^ { 0 } \in \mathbb { R } , \xi \in \mathbb { R } ^ { d } \}$ is called the space of replicablepayoffs.

# Derivatives I

# Definition

Assume that $\mathcal { F } _ { 1 } = \sigma ( S _ { 1 } ^ { 0 } , \ldots , S _ { 1 } ^ { d } )$ . A derivative of payoff $H$ is a function ofthe prices $S _ { 1 } ^ { 0 } , \ldots , S _ { 1 } ^ { d }$ , i.e.

$$
H = f \left(S _ {1} ^ {0}, \dots , S _ {1} ^ {d}\right)
$$

for some measurable function f : Rd+1 → R.

Forward contract : We call a forward contract on the asset $S ^ { j }$ theagreement at $t = 0$ between the buyer and the seller s.t. the buyerwill buy at $t = 1$ one unit of the asset $S ^ { j }$ with the fixed price K (attime 0). The corresponding payoff is therefore written

$$

$$

# Derivatives II

▶ Call option : We call a call option on the asset $S ^ { j }$ the agreement at$t = 0$ between the buyer and the seller s.t. the buyer has the right,but not the obligation, to buy at $t = 1$ one unit of the asset $S ^ { j }$ withthe fixed price K (called strike and decided at time 0). Thecorresponding payoff is therefore written

$$
H ^ {C} = \max  \left(S _ {1} ^ {i} - K, 0\right) = \left(S _ {1} ^ {i} - K\right) ^ {+}.

▶ Put option : Similarly,

$$

$$

▶ Basket option, straddle, butterfly spread, etc.

# Definition

A market is said to be complete if all the payoffs are replicable.

# Theorem
